## Run Locally

Clone the project

```bash
  git clone git@github.com:Antonwy/mvst-github-api-challenge.git
```

Go to the project directory

```bash
  cd mvst-github-api-challenge
```

Install dependencies

```bash
  npm install
```

Start the NextJS server

```bash
  npm run dev
```

Start the Storybook server

```bash
  npm run storybook
```
